# Library Dog

A self-hosted book discovery + download tool. Punch in an author, pick from
their bibliography, and Library Dog finds and grabs EPUBs into a flat folder
your Calibre / Calibre-Web-Automated / Komga library can ingest.

> **Heads up:** this is largely Claude-assisted ("vibe-coded") code derived from
> a personal project. It works well for the author's setup but should be
> treated as alpha-quality if you're running it yourself. Read the diff before
> pointing it at anything sensitive, and expect rough edges.

The README is filled in as the public-ready cut lands. For now, see commit
history for what's wired up and what isn't.
